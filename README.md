Wordle Game

A Wordle-style game developed with **Qt 6**, **C++** and **QML**.  
Users try to guess the 5-letter target word within limited attempts.  
The game shows feedback for correct letters and positions, warns for invalid or duplicate guesses,  
and displays a dialog on win or loss.


*It checks if word is long enough or have a meaning in dictionary.
